from tensorflow.keras.layers import *


model = Sequential([

Conv2D(

(3,3),

Conv2D(

# 4. CNN + LSTM pentru video

Aici este mai interesant.

Nu mai vezi frame-ul separat.

Vezi:

```
video.mp4

↓

[
frame1,
frame2,
frame3,
...
frame20
]

↓

CNN extrage feature

↓

LSTM învață evoluția temporală
```

Structura:

```
Video
 |
 |
20 frames
 |
 |
CNN
 |
 |
LSTM
 |
 |
Fake/Real
```

Model:

```python
from tensorflow.keras.layers import *


model = Sequential([

TimeDistributed(
Conv2D(
32,
(3,3),
activation="relu"
),
input_shape=(20,224,224,3)
),

TimeDistributed(MaxPooling2D()),


TimeDistributed(
Flatten()
),


LSTM(128),


Dense(
1,
activation="sigmoid"
)

])
```

Input:

```
(batch,20,224,224,3)
```

# 5. GAN detection

Aici trebuie să clarificăm ceva.

Un GAN detector nu înseamnă neapărat că antrenezi un GAN.

În deepfake detection, GAN-based poate însemna:

* extractor de features
* discriminator
* model antrenat pe imagini generate

Structură:

```
Real images
Fake images
      |
      |
Generator features
      |
      |
Discriminator
      |
      |
Fake probability
```

Pentru GAN ai nevoie de:

```
generator.py
discriminator.py
```

sau poți folosi un encoder:

```
EfficientNet
      |
features
      |
classifier
```

De fapt modelul din repo:

```
EfficientNet-B0
```

este deja o abordare modernă de discriminator.
